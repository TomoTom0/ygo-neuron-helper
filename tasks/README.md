## tasks

tasks/でタスクを管理する。

## files

- README.md: このファイル。閲覧用。
- todo.md: 実行予定のタスクについて記載する。原則完了済みの内容は記載しない。(実行予定の内容と密接に関係する場合のみ記載してもよい)
- wip.md: 現在進行中のタスクについて記載する。完了したらtodo.mdに移動する。より詳細に記載したい場合はtmp/wip/に個別にファイルを作成する。
- done.md: 完了したタスクについて記載する。完了日時や関連PRなども記載する。ファイルが肥大化したらdocs/_archived/に複製して、done.mdには最近の内容だけ残す。
- milestone.md: マイルストーンごとの主要タスクを記載する。基本的に参照用であり、実行予定や実行中などの情報は上記のファイルに記載する。
- pending.md: 保留中のタスクについて記載する。将来的に再検討する可能性があるが、現時点では実行しないタスクを記載する。
