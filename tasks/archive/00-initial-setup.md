# 00. 初期セットアップタスク

## 目的
プロジェクトの基本構造と開発環境を整える

## タスク

### プロジェクト構造
- [ ] プロジェクトのディレクトリ構造を決定
- [ ] TypeScript/Node.jsの初期化（package.json作成）
- [ ] 必要な開発依存関係のインストール（TypeScript, ESLint, Prettierなど）
- [ ] tsconfig.jsonの設定
- [ ] ビルド設定の構築

### Chrome拡張機能セットアップ
- [ ] manifest.jsonの作成（Manifest V3）
- [ ] 基本的なディレクトリ構造の作成
  - [ ] src/（ソースコード）
  - [ ] public/（静的ファイル）
  - [ ] dist/（ビルド出力）
- [ ] 開発用のビルドスクリプト設定
- [ ] Hot reloadの設定（開発効率化）

### Git/バージョン管理
- [ ] .gitignoreの作成
- [ ] README.mdの作成（プロジェクト説明）
- [ ] 初期コミット

## 完了条件
- プロジェクトが正常にビルドできる
- Chrome拡張機能として最小限の動作が確認できる
