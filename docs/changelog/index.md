# Changelog

遊戯王NEXT (遊戯王 Neuron EXTension) の全バージョンの変更履歴です。

## バージョン一覧

- **[v0.4.0](./v0.4.0.md)** - 2025-11-22 - **最新版**
  - 検索・フィルター機能の強化（SearchInputBar共通化、グローバル検索モード）
  - ソート機能拡張（7種類のソートオプション）
  - カテゴリ/タグ自動分類機能
  - UI/UX改善（Load Dialogリデザイン、設定ダイアログ2x2グリッド化）
  - テスト全面修正（370件全て通過）

- **[v0.3.1](./v0.3.1.md)** - 2025-11-14
  - popup CSS分離とCSS変数導入
  - webpack修正（致命的バグ修正）
  - E2Eテスト実施（カード詳細・表示モード・ソート機能）
  - ブランチ保護強化

- **[v0.3.0](./v0.3.0.md)** - 2025-11-13
  - 独自デッキ編集画面の実装
  - 多言語対応基盤の整備
  - オプションページの拡張

- **v0.2.0** - 2025-11-10
  - デッキ画像作成機能
  - シャッフル・ソート・固定機能
  - オプションページによる機能制御
  - 画像ベースの効果タイプ判定

- **v0.1.0** - 初期リリース
  - 基本的なAPI実装
  - デッキ詳細取得
  - カード情報取得

### v0.3.0 - デッキ編集UI + 多言語対応

  - API多言語対応
- **オプションページ拡張**
  - デッキ編集機能の設定
  - 多言語設定

- UI/UX改善（レスポンシブ、アニメーション）
