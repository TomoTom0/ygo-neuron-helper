# Changelog

Yugioh Neuron Helper の全バージョンの変更履歴です。

## バージョン一覧

- **[v0.3.0](./v0.3.0.md)** - 2025-11-13 - **最新版**
  - 独自デッキ編集画面の実装
  - 多言語対応基盤の整備
  - オプションページの拡張

- **v0.2.0** - 2025-11-10
  - デッキ画像作成機能
  - シャッフル・ソート・固定機能
  - オプションページによる機能制御
  - 画像ベースの効果タイプ判定

- **v0.1.0** - 初期リリース
  - 基本的なAPI実装
  - デッキ詳細取得
  - カード情報取得

## バージョン別の主要機能

### v0.3.0 - デッキ編集UI + 多言語対応

#### 新機能
- **独自デッキ編集画面** (#/ytomo/edit)
  - DNO入力によるデッキロード
  - カード検索・詳細表示
  - ドラッグ＆ドロップ
  - カード枚数制限（3枚まで）
- **多言語対応基盤**
  - 言語自動検出
  - マッピングテーブル
  - API多言語対応
- **オプションページ拡張**
  - デッキ編集機能の設定
  - 多言語設定

#### 改善
- UI/UX改善（レスポンシブ、アニメーション）
- コード品質改善（型定義、コンポーネント分離）
- テスト実装（125 tests）

#### 既知の問題
- デッキ保存ダイアログは未完成
- E2Eテストは未実装

詳細: [v0.3.0 Changelog](./v0.3.0.md)

---

### v0.2.0 - 独立機能の実装

#### 新機能
- **デッキ画像作成機能**
  - Canvas描画、QRコード生成
  - カラーバリエーション（赤/青）
  - ダイアログUI
- **シャッフル・ソート・固定機能**
  - メインデッキのシャッフル
  - 元の順序への復元
  - カード固定（ピン留め）
- **オプションページ**
  - 各機能のON/OFF切り替え
  - 機能説明・使い方表示

#### 改善
- 画像ベースの効果タイプ判定（言語非依存）
- 型安全性とVue互換性のバランス調整

---

### v0.1.0 - 初期リリース

#### 新機能
- **基本API実装**
  - デッキ詳細取得（getDeckDetail）
  - デッキ一覧取得（getDeckList）
  - カード詳細取得（getCardDetail）
  - カードFAQ取得（getCardFAQList, getFAQDetail）
- **HTMLパーサー実装**
  - カード詳細情報のパース
  - デッキ情報のパース
  - FAQ情報のパース
