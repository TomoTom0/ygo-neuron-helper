# オプションページガイド

Chrome拡張機能のオプションページでの設定項目について説明します。

## アクセス方法

1. Chromeの拡張機能アイコンをクリック
2. 「オプション」または歯車アイコンをクリック
3. または、chrome://extensions から「詳細」→「拡張機能のオプション」

## タブ構成

### 1. Omit and Usage タブ

各機能のON/OFF切り替えと使い方の説明。

#### シャッフル機能
- **説明**: デッキ表示ページでカードの順序をランダム化
- **ON/OFF**: 機能の有効/無効を切り替え
- **デモ**: GIFアニメーションで使い方を表示

#### デッキ画像作成機能
- **説明**: デッキレシピを画像として保存
- **ON/OFF**: 機能の有効/無効を切り替え
- **サンプル**: 作成される画像の見本を表示

#### 独自デッキ編集画面
- **説明**: 拡張されたデッキ編集UI
- **ON/OFF**: 機能の有効/無効を切り替え
- **アクセスURL**: `https://www.db.yugioh-card.com/yugiohdb/#/ytomo/edit`

### 2. Deck Edit Settings タブ

デッキ編集画面の詳細設定。

#### 表示設定

**デフォルト表示モード**
- **リスト**: カード名と情報を一覧表示
- **グリッド**: カード画像をタイル表示

**デフォルトソート順**
- 発売日順
- 名前順
- ATK順
- DEF順
- Lv/Rank順
- 属性順
- 種族順

#### アニメーション設定

**アニメーション有効化**
- **ON**: カード移動時にFLIPアニメーションを表示
- **OFF**: アニメーションなしで即座に移動

#### 検索設定

**検索バーの位置**
- **デフォルト**: 各セクションに検索バーを表示
- **グローバル**: 画面上部に固定表示

#### 言語設定

**言語選択**
- **自動**: ページの言語を自動検出
- **日本語**: 常に日本語で表示
- **English**: 常に英語で表示

## 設定の保存

設定は自動的にlocalStorageに保存されます。ブラウザを閉じても設定は維持されます。

### 設定のリセット

「デフォルトに戻す」ボタンで全ての設定を初期値に戻せます。

## 機能別の推奨設定

### 初心者向け
- 表示モード: リスト
- アニメーション: ON
- 言語: 自動

### 上級者向け
- 表示モード: グリッド
- アニメーション: OFF（高速操作）
- ソート順: 名前順

### モバイル使用時
- 表示モード: グリッド（画面効率）
- アニメーション: OFF（パフォーマンス）

## トラブルシューティング

### 設定が保存されない
1. ブラウザのlocalStorageが有効か確認
2. プライベートブラウジングモードでは保存されない場合があります
3. 拡張機能を再インストール

### 機能が動作しない
1. Omit and Usageタブで機能がONになっているか確認
2. ページをリロード
3. 拡張機能を無効化→有効化

### 言語設定が反映されない
1. 自動検出が正しく動作しているか確認
2. 手動で言語を選択
3. ページをリロード
