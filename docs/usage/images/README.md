## 撮影時の注意事項

- すべての画像は tests/sample/url.md に記載された公開デッキURLを使用
- 画像フォーマット: PNG
- 解像度: 実際のブラウザ表示サイズ
- ボタンやUI要素が明確に見えるように撮影
