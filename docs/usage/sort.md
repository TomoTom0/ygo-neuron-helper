# ソート機能ガイド

デッキ編集画面でのカードソート機能の使い方を説明します。

## 概要

v0.4.0でソート機能が大幅に拡張されました。7種類のソートオプションから選択でき、カードを様々な基準で並び替えることができます。

## ソートオプション

### 1. 発売日順（デフォルト）
カードの発売日順に並び替えます。最新カードが先頭になります。

### 2. 名前順
カード名の50音順（日本語）またはアルファベット順（英語）で並び替えます。

### 3. ATK順
攻撃力の高い順に並び替えます。モンスターカードのみ対象です。

### 4. DEF順
守備力の高い順に並び替えます。モンスターカードのみ対象です。

### 5. Lv/Rank順
レベル・ランクの高い順に並び替えます。
- 通常モンスター、効果モンスター: レベル
- エクシーズモンスター: ランク
- リンクモンスター: リンク数

### 6. 属性順
属性でグループ化して並び替えます。
順序: 闇 → 光 → 地 → 水 → 炎 → 風 → 神

### 7. 種族順
種族でグループ化して並び替えます。
順序: 戦士族 → 魔法使い族 → ドラゴン族 → ...（50音順）

## 使い方

### 検索結果のソート

1. 検索バーの右側にあるソートボタンをクリック
2. ドロップダウンからソート順を選択
3. カードリストが即座に並び替えられる

### ソート方向の切り替え

同じソートオプションを再度クリックすると、昇順/降順が切り替わります。

- 昇順: 小さい値から大きい値へ
- 降順: 大きい値から小さい値へ

## デフォルトソート順の設定

設定ダイアログで、デフォルトのソート順を変更できます。

**設定方法**:
1. 設定ダイアログを開く（歯車アイコン）
2. 「デフォルトソート順」を選択
3. 希望のソートオプションを設定

## ソートの適用範囲

### 検索結果リスト
検索・フィルター結果の表示順に適用されます。

### デッキセクション
デッキ内のカード表示順には直接影響しません。デッキ内のカード順序は手動で管理します。

## Tips

### 効率的なソートの使い方

1. **デッキ構築時**: 名前順でカードを探しやすく
2. **バランス確認**: Lv/Rank順でレベル分布を確認
3. **火力確認**: ATK順で攻撃力を比較
4. **属性統一デッキ**: 属性順で属性ごとにグループ化

### ソートとフィルターの組み合わせ

フィルターで絞り込んだ後にソートを適用すると、より効率的にカードを探せます。

例：
1. 「レベル4以下」でフィルター
2. 「ATK順」でソート
3. 攻撃力の高い低レベルモンスターを確認
