# プライバシー & テレメトリレビュー依頼書

**作成日時**: 2025-11-18
**対象**: ログ収集、オプトイン/オプトアウト、ユーザーデータ処理フロー
**レビュー目的**: ユーザーデータの取り扱いの妥当性確認とコンプライアンス対応
**レポート出力先**: `docs/internal-reviews/reports/privacy-telemetry.md`

## レビュー依頼項目

- A. 収集データの洗い出し
  - 対象: ログ、エラー報告、解析イベント、外部サービスへの送信
  - 確認事項: 収集しているフィールドとそれが個人情報に該当するか

- B. ユーザー同意の扱い
  - 確認事項: 明示的な同意・通知・オプトアウト機構の有無

- C. データ保持と削除ポリシー
  - 確認事項: 保持期間、削除手順、バックアップ方針

- D. 外部サービス連携のレビュー
  - 確認事項: サードパーティへの送信先、暗号化、契約上の要件

## 期待する成果物

- `docs/internal-reviews/reports/privacy-telemetry.md` - 収集項目一覧、リスク評価、改善提案
- 必要なプライバシーポリシー記載の差分案（例: README / docs への追加文言）
