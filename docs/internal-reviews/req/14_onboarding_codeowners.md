# コントリビュータ導入 & 所有権レビュー依頼書

**作成日時**: 2025-11-18
**対象**: `CODEOWNERS`, `CONTRIBUTING.md`, ドキュメント化されていない運用手順
**レビュー目的**: 新規コントリビュータの導入を円滑にし、所有権を明確にする
**レポート出力先**: `docs/internal-reviews/reports/onboarding-codeowners.md`

## レビュー依頼項目

- A. CODEOWNERS の妥当性確認
  - 確認事項: 主要領域の所有者設定、レビュープロセスとの整合性

- B. CONTRIBUTING / 開発環境セットアップ手順の整備
  - 確認事項: ローカルビルド、テスト実行、デバッグ手順の明瞭さ

- C. PR / Issue テンプレートの整備
  - 確認事項: 必須情報の網羅、レビューチェックリストの有無

- D. オーナーシップによる責任範囲の明示
  - 確認事項: ドメイン毎の担当者候補、オンコール・緊急対応の流れ

## 期待する成果物

- `docs/internal-reviews/reports/onboarding-codeowners.md` - 提案される `CODEOWNERS` と `CONTRIBUTING.md` の草案
- PR / Issue テンプレートのサンプル
